Welcome to Ceneo Web Scraper

This project is a web scraper for extracting product opinions from Ceneo.

Purpose of the application

To automate extraction and analysis of product reviews from Ceneo.pl

Description of the application's operation

User inputs a product ID, app scrapes reviews, stores them in JSON files, and allows viewing and downloading summaries.

Libraries used

  • Flask
  • Requests, BeautifulSoup (for scraping)
  • Pandas
  • Deep Translator

Applied approaches

  • Web scraping
  • Data analysis with Pandas
  • Flask for the web interface
